### 数组去除

思路:

利用双循环: 外循环i,内循环j

内循环j的初始值为 `i+1 = arr[i]` 相邻的数，即是内循环一开始，便从切片第二个索引与 `arr[i]` 进行对比。

```go
package main

import "fmt"

func RemoveRepeatSlice(s []int) (newSlice []int) {

	newSlice = make([]int, 0) //初始化返回值

	for i := 0; i < len(s); i++ {

		repeat := false //建立标志

		for j := i + 1; j < len(s); j++ {
			if s[i] == s[j] { //判断外循环索引值是否与内循环的每个索引值相等
				repeat = true
			}
		}

		//每一轮内循环会循环len(s)次，如果repeat为true，说明s[i]与内循环其中某值相等
		if !repeat {
			newSlice = append(newSlice, s[i])
		}
	}
	return newSlice
}

func main() {

	s := []int{2, 1, 45, 2, 1, 6, 3, 3, 7, 9, 0, 1}
	newSlice := RemoveRepeatSlice(s)
	fmt.Println(newSlice)      //[45 2 6 3 7 9 0 1]
	fmt.Println(cap(newSlice)) //8

}


```
