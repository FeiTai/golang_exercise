### 数组找出第一大的数和第二大的数

整体思路是:

max的值必须为数组第一位数

每次对比两个相邻的数，来筛选出最大数，和第二大的数

`len(arr) -1`是为了每次对比结整就减少一次循环次数

```go
package main

import (
	"fmt"
)

func findMaxAndSecond(arr []int) (max, second int) {
	max = arr[0] //max初始值为切片索引第一位

	for i := 0; i < len(arr)-1; i++ {

		if max < arr[i+1] {
			second = max
			max = arr[i+1]
		} else {

			if second < max {
				if second < arr[i+1] {
					second = arr[i+1]
				}
			}

		}

	}

	return max, second
}

func main() {

	arr := []int{1, 23, 55, 11, 44, 65, 32, 77}
	max, second := findMaxAndSecond(arr)
	fmt.Printf("最大值为:%d\n", max)
	fmt.Printf("第二大值为:%d\n", second)

}

```
