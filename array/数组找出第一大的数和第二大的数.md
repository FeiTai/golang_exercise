### 数组找出第一大的数和第二大的数

整体思路是:

max的值必须为数组第一位数

通过对比每个数,计算出最大值和第二最大值

```go
package main

import (
	"fmt"
)

func findMaxAndSecond(arr []int) (max, second int) {
	max = arr[0] //max初始值为切片索引第一位

	for i := 0; i < len(arr); i++ {

		if max < arr[i+1] {
			second = max
			max = arr[i+1]
		} else {

			if second < max {
				if second < arr[i+1] {
					second = arr[i+1]
				}
			}

		}

	}

	return max, second
}

func main() {

	arr := []int{1, 23, 55, 11, 44, 65, 32, 77}
	max, secnd := findMaxAndSecond(arr)
	fmt.Printf("最大值为:%d\n", max)
	fmt.Printf("第二大值为:%d\n", secnd)

}

```
