### 顺序函数过滤容器

用顺序函数过滤容器：s 是前 10 个整型的切片。构造一个函数 Filter，第一个参数是 s，第二个参数是一个 fn func(int) bool，返回满足函数 fn 的元素切片。通过 fn 测试方法测试当整型值是偶数时的情况。

所用知识:高阶函数(将函数当作参数传给函数)

```go
package main

import "fmt"

func Filter(s []int, fn func(int) bool) {
	for i := 0; i < len(s); i++ {
		if !fn(i) {
			fmt.Printf("%d is not even!!", i)
		} else {
			fmt.Printf("%d is even!!!\n", i)
		}
	}

}

func isEven(n int) (f bool) {
	//取余计算
	if n%2 == 0 {
		return true
	}

	return false
}

func main() {
	s := make([]int, 0)
	for i := 0; i < 50; i++ {
		s = append(s, i)
	}

	Filter(s, isEven)
}


```
