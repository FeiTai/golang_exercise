### 冒泡排序

思路:每次循环比较两个相邻的数，直到循环结束

`len(s)-1`是为了优化最后不必要的对比

```go
package main

import "fmt"

func BubbleSort(s []int) {
	for i := 0; i < len(s)-1; i++ {
		for j := 0; j < len(s)-i-1; j++ {

			if s[j] > s[j+1] {
				s[j], s[j+1] = s[j+1], s[j] //swap value
			}
		}
	}
}

func main() {
	arr := []int{9, 8, 7, 6, 5, 4, 3, 2, 1}
	BubbleSort(arr)
	fmt.Println(arr)
}

```
